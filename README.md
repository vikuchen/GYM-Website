# IronForge Gym • Premium Fitness Framework Portfolio

Repositori rancangan website landing page satu halaman modern berperforma tinggi yang didedikasikan untuk platform industri kebugaran komersial (Gym & Pusat Kebugaran). Dirancang menggunakan kombinasi Bootstrap 5 Utility Engine dan standarisasi arsitektur multi-file modern.

---

## 💎 Fitur Utama Tata Kelola Sistem

* **Pemisahan Kode Berkas Kosmetika (Separation of Concerns):** Seluruh file dipisah secara fungsional (HTML5 sebagai markah data struktur, CSS3 selaku penata visual tata letak, dan ES6 JS mengolah fungsi interaktivitas).
* **Komponen Toast Efisien:** Mesin skrip menginjeksikan pesan feedback interaktif secara asinkronus ke dalam ruang DOM tanpa membutuhkan dependensi library tambahan.
* **Sistem Navigasi Adaptif (Smooth Navigation Layer):** Hubungan navigasi antar-konten internal dikelola menggunakan animasi halus terpadu untuk memberikan impresi pameran digital yang nyaman (UX).
* **Optimalisasi Breakpoint Seluler:** Penataan responsif penuh mengandalkan fluid grid yang kokoh untuk menjamin integrasi antarmuka layar sentuh di ponsel pintar (Android/iOS) maupun tablet.

---

## 🛠️ Pustaka & Teknologi yang Digunakan

* **Bootstrap v5.3.0** – Sistem responsif grid layouting framework.
* **FontAwesome Icons Grid** – Integrasi aset pustaka ikon vektor grafis.
* **Google Fonts API Engine** – Pengolah tipografi modern berbobot tinggi.
* **JavaScript ES6 Vanilla Architecture** – Modul enkapsulasi fungsi terpadu IIFE.
